# Julia Price Portfolio

A personal portfolio site built with React and Vite.

## Features
- Home page with hero section and background image
- About page with bio, skills badges, and interests
- Projects page with image cards, live demo and GitHub links
- Contact page with email, GitHub, and LinkedIn cards
- Responsive layout across mobile, tablet, and desktop
- Framer Motion page animations


## Tech Stack

- React 19, Vite, React Router v7
- Bootstrap 5, Framer Motion

## Run Locally

```bash
npm install
npm run dev
```
